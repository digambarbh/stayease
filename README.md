# StayEase 🏨

A premium, fully responsive, and feature-rich hotel & property booking platform.

## 📖 About The Project

StayEase is a modern, full-stack web application designed to connect travelers with hosts offering unique accommodations. Built with performance and aesthetics in mind, it provides a seamless booking experience complete with secure payments, interactive maps, and comprehensive dashboards for Users, Hosts, and Administrators.

### ✨ Key Features

*   **Secure Authentication:** Role-based access control with distinct experiences for Users, Hosts, and Admins.

*   **Interactive Maps:** Real-time location mapping powered by MapTiler.
*   **Cloud Storage:** High-performance image uploading via Cloudinary.
*   **Dedicated Dashboards:** Centralized monitoring and user moderation for Admins, earnings tracking for Hosts, and booking history for Users.
*   **Advanced Search & Filters:** Filter properties by type, price range, amenities, and ratings.
*   **Fully Responsive:** Premium, mobile-first design utilizing sleek UI elements.

## 🛠️ Built With

*   **Frontend:** HTML5, CSS3, EJS, Bootstrap 5
*   **Backend:** Node.js, Express.js
*   **Database:** MongoDB, Mongoose ODM
*   **Authentication:** JSON Web Tokens (JWT)
